\abstract{This is an example poster using the ISG open day beamer theme.  The \texttt{.tex} file should be a good guide to help people to use the theme.  The poster gives some description of the few things that authors will need to be aware of.  Generally it is expected that anyone familiar with \LaTeX\ should be able to use this poster style without difficulty.  I think that the style is fairly stable, however I doubt it's perfect, if you find an error please let me (Rob Lee) know via git or by email.}
\problemDescription{Deciding on what style to use to create a nice \LaTeX\ poster for the RHUL Information Security Group Open Day.}
\solutionDescription{Using the beamer style file created for use at the RHUL Information Security Group Open Day.  With this style file you can painlessly create a nice looking poster to present at the open day.  Here is presented an example poster that shows off this style.}
